# 香煎鸡胸肉饼

![香煎鸡胸肉饼](../Images/香煎鸡胸肉饼.jpg)

## 材料

| 一人份 |       |
| ------ | ----- |
| 鸡胸   | 250克 |
| 西蓝花 | 80克 |
| 香菇 | 2朵 |
| 胡萝卜 | 80克 |
| 甜玉米粒 | 适量 |
| 盐 | 1茶匙 |
| 生抽 | 2汤匙 |
| 蚝油 | 1汤匙 |
| 黄酒 | 2汤匙 |
| 淀粉 | 少许 |

## 做法

1. 将所有蔬菜焯水。
2. 将鸡胸肉斩成稍带颗粒感的泥，蔬菜切米粒丁；也可直接用料理机处理。
3. 取一盆。将鸡胸肉、蔬菜、调料搅匀成上劲的肉馅。
4. 平底锅入少许油。取约50克肉馅，拍成饼状下锅煎制。
5. 煎成两面金黄后装盘。

## 注记

- 可以替换成其它蔬菜的组合，如青豆、芹菜等。
